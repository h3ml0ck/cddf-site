# cddf-site

Marketing and documentation site for the [Citizen Drone Defense Force](https://github.com/h3ml0ck/cddf) project. Built with React, Vite, and Tailwind CSS.

## Development

Install dependencies:

```bash
npm install
```

Start the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

Lint:

```bash
npm run lint
```

## Stack

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Lucide React](https://lucide.dev/)

## Configuration

Update `GITHUB_URL` in `src/App.jsx` if the repository URL changes.

Update the `<loc>` in `public/sitemap.xml` with the production domain once it is known.
